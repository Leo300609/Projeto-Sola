# 🎸 Tribute Page: Projeto Sola

Este projeto é uma página de tributo dedicada à banda brasileira Projeto Sola. O objetivo foi criar uma interface informativa e visualmente harmoniosa que conte a história, a fundamentação teórica (Cinco Solas) e as curiosidades do duo de folk cristão.

## 🚀 Tecnologias Utilizadas
* **HTML5:** * **Semântica:** Uso de `<figure>` para imagens e `<figcaption>` para legendas.
    * **Citações:** Implementação da tag `<blockquote>` para destacar letras de músicas e versículos.
* **CSS3:**
    * **Paleta de Cores Customizada:** Uso de tons de marrom e bege para refletir a estética acústica/folk da banda.
    * **Tipografia:** Estilização de títulos e textos enfatizando termos importantes com classes de negrito.
    * **Bordas Estilizadas:** Uso de `border-top: 1px dashed` para criar divisões suaves e elegantes entre as seções.

## 🧠 Lógica e ADS Aplicada:
* **Hierarquia de Informação:** Organização clara entre Cabeçalho, Imagem de Destaque, Biografia, Citações e Curiosidades.
* **Layout Responsivo:** Imagens configuradas com `max-width: 100%` para garantir que a página não quebre em telas menores.
* **Design Contextual:** Escolha de cores e fontes que conversam diretamente com o tema do projeto, focando na experiência do usuário e na leitura agradável.

## ⚙️ Como visualizar
1. Clone este repositório.
2. Abra o arquivo `index.html` no seu navegador.
